Comando = nombre de programa +  flags + argumentos

Flags: alteran lo que va a hacer el programa
Argumentos: Informacion adicional para la ejecucion del programa


ej: comando -flag1 -flag2 arg1 arg2





Mostrar el manual de un comando, por ejemplo para echo:
$ man echo
